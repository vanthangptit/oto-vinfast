# Use NodeJS v18.19.0

# To run dev
`1. npm i`

`2. npm i -g gulp-cli`

`3. npm run dev & gulp dev`

### Now, app will bootstrap with port in config. Default is 8080

# To use public source
`/[scripts, styles]`

#Edit file `.env.example` to `.env`

## Run DOCKER
`docker-compose up --build`
### Now, app run on port `9100`
